# stock-cli

## start
### install 
`npm install -g jd-stock-cli`

### run 
```
➜  ~ stock aapl
┌────────────┬──────────┬──────────┬────────┐
│ 股票名称     │ 当前股价  │ 当前市值  │ 市盈率  │
├────────────┼──────────┼──────────┼────────┤
│ Apple Inc. │ 319.00   │ 13957.78 │ 25.33  │
└────────────┴──────────┴──────────┴────────┘
```
